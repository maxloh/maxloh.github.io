About Me

My name is Loh Ka Hong, a current Year 3 student of The Open University of Hong Kong enrolled in the undergraduate programme, Bachelor of Computing with Honours in Internet Technology.

I have particular interest in programming languages which I find them mind challenging with sense of fulfilment through the process. I spend my free time to explore more programming languages other than what I have taught in classes, like Python and JavaScript. I love to browse GitHub and look into source code of interesting projects to understand their usages, even though I may not know its programming language.

I am an enthusiast of front-end development. It is like an achievement to use what I design and built. Being a fan of user interface design, I investigate different user interface software to inspect their content organization and implementation of visual hierarchy. I also study web technology and UI design online.

I studied basics of php in my database course which I implemented in my group projects. The course also aroused my interests in back-end development which I did research on Web, and have some basic concepts on related technology like Note.js and different types of database.

Projects

OUHK CGPA Calculator Star

OUHK CGPA Calculator is a a few lines of JavaScript code to help OUHK students calculate their CGPA. The calculater can be run directly in Chrome DevTools or used as a JavaScript bookmarklet. It automatically search for the respective table containing reuslts, parse the table to json varieable and calculate the CGPA. Than display the CGPA using the alert() function.

Secondary school website

This website is written for the group project of my web accessibility course in college. In the project, I in charged of the design and coding of the website and my teammate was responsible for documentation. The final product was powered by Bootstrap. Contents of the website was reorganized into different sections and presented in a more user-friendly way. Different components like alternative text and ‘skip to contents’ navigation was added to the website to make it more accessible to people with disability.
